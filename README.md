# 💰 Personal Finance AI Advisor (India)

A simple and intelligent personal finance tracker built using Python and Streamlit.
This application helps users track daily expenses, analyze spending habits, and get basic AI-driven financial insights.

---

## 🚀 Features

* 📌 Add and manage daily expenses
* 📊 Category-wise spending visualization
* 🧠 AI-based spending insights
* 📈 Monthly expense prediction
* 🇮🇳 India-focused categories (UPI, Rent, Bills, etc.)
* 🖥️ Clean and interactive web interface

---

## 🛠️ Tech Stack

* Python
* Streamlit
* Pandas

## ▶️ How to Run Locally

1. Clone the repository:

```
git clone https://github.com/your-username/finance-ai-advisor.git
cd finance-ai-advisor
```

2. Install dependencies:

```
pip install -r requirements.txt
```

3. Run the app:

```
streamlit run app.py
```

4. Open in browser:

```
http://localhost:8501
```

## 🧠 How It Works

* Users input daily expenses through a simple UI
* Data is stored using Streamlit session state
* The app analyzes spending patterns by category
* Generates insights based on spending thresholds
* Predicts future expenses using average spending
